Projet VR – Duel Médiéval
Présentation

Ce projet de réalité virtuelle a été réalisé en groupe sur CoSpaces Edu.
On a imaginé un univers médiéval dans lequel l’utilisateur peut se déplacer et interagir avec différents éléments de la scène. Le but était de créer une petite expérience immersive avec plusieurs interactions simples mais efficaces.

Fonctionnalités techniques

Navigation :  On a mis en place un système de 6 points de téléportation permettant de franchir la porte, d'observer le duel sous deux angles, et de monter dans les tribunes pour rejoindre le public.

Feedback sonore : Un son se déclenche à chaque téléportation pour montrer que l’action a bien été prise en compte.

Interaction avec le premier chevalier :  Quand on clique dessus, un panneau apparaît avec des informations pour orienter le joueur.

Interaction avec le deuxième chevalier : Cette fois, on a choisi un effet visuel : le chevalier change de couleur et des particules apparaissent pour rendre l’interaction plus dynamique.
